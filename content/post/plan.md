### 要不要部署到netlify？

C指导回答：

>| 对比项                 | GitHub Pages              | Netlify                     |
| ---------------------- | ------------------------- | --------------------------- |
| 是否需要 GitHub        | **必须**（源码在 GitHub） | **不必须**（但常用 GitHub） |
| Hugo 构建方式          | 本地构建或 GitHub Actions | Netlify 自动构建            |
| 是否需要 CI（Actions） | **通常需要**              | **不需要**                  |
| 构建速度               | 一般                      | **快**                      |
| 自定义功能             | 较少                      | **非常丰富**                |
| 免费额度               | 完全免费                  | 免费版足够个人博客          |
| CDN / HTTPS            | 有                        | **更强**                    |
| 自定义域名             | 支持                      | 支持（配置更简单）          |
